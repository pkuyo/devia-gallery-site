# Devia Gallery Site

Static public pages for Devia Gallery.

## Privacy Policy URL

After publishing this repository with GitHub Pages, the privacy policy URL will be one of:

- `https://<github-username>.github.io/devia-gallery-site/privacy.html`
- `https://<github-username>.github.io/privacy.html` if this repository is named `<github-username>.github.io`

## GitHub Pages Setup

1. Create a public GitHub repository, for example `devia-gallery-site`.
2. Push this folder to the repository.
3. In GitHub, open `Settings` -> `Pages`.
4. Set source to `Deploy from a branch`.
5. Select branch `main` and folder `/root`.
6. Save, then wait for GitHub Pages to publish.

No build step is required.
